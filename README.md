#  Home Cyber Security Lab (SOC Simulation)

This project simulates a basic Security Operations Center (SOC) environment for learning and practicing cybersecurity monitoring, scanning, and analysis.



##  Objective
To understand how security analysts detect, monitor, and respond to potential threats in a network environment.



##  Key Activities

###  Network Scanning
 Performed using Nmap
 Identified open ports and services

###  Traffic Analysis
 Captured network packets using Wireshark
 Analyzed suspicious traffic patterns

###  System Monitoring
 Checked running processes in Linux
 Identified active services and system behavior


## 🛠️ Tools Used
 Nmap
 Wireshark
 Linux terminal
 Python (basic automation scripts)
